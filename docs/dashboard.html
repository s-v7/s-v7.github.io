Overview

CREAPI Analytical Platform is an institutional analytics and decision-support system designed for CREA environments, integrating operational data, financial intelligence, ART analytics, ML prediction and governance tooling.

It is not a simple dashboard — it is a modular analytical platform combining Streamlit, ML pipelines, OCR, RAG, automation and audit layers.

Functional Modules

  • ART analytics and lifecycle metrics
  • Financial & delinquency intelligence
  • Professional & company analytics
  • Inspection & violation panels
  • Protocol & workflow monitoring
  • Demand routing & sector workload
  • OCR document classification
  • RAG normative assistant panels
  • Blockchain event audit trail

Machine Learning Layer

  • ART analysis time prediction
  • Delinquency prediction models
  • Boleto payment forecasting
  • Risk scoring models
  • Association rules mining
  • Anomaly detection

Models are versioned and trained through controlled pipelines. Predictions are decision-support only.

Automation & Jobs

  • Scheduled ML retraining
  • Automated alerts & nudges
  • Weekly reports generation
  • Database backup jobs
  • Kubernetes cronjobs
  • Ansible deployment automation

Infrastructure Stack

  • Python 3.12 + Streamlit
  • PostgreSQL analytics DB
  • Docker containers
  • Kubernetes workloads
  • Ansible orchestration
  • Terraform provisioning

Governance & Audit

  • Event logging
  • Analytical audit trails
  • Transparent ML usage
  • Institutional explainability
  • Sector accountability metrics

Private Codebase — Public Technical Documentation

Source code, database connectors and operational logic are private due to institutional security and regulatory requirements. This documentation describes architecture, scope and analytical capabilities only.
