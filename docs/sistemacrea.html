Overview

SIGEC is a corporate platform developed for CREA-PI to manage processes, protocols, financial, registry and institutional workflows.

Technology Stack

Java 8 • Jakarta EE • JSF • PrimeFaces • PostgreSQL • Payara • FastAPI integrations
