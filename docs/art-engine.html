Overview

ART Intelligence Engine is a modular platform for technical and semantic analysis of ART documents, designed to support CREA analysts in validation, audit and interpretation workflows.

The system combines deterministic rules, institutional database crosscheck, PDF extraction and LLM-assisted semantic enrichment — always as decision-support, never automated decision.

Architecture Modules

  • ai/analysis — deterministic rule engine
  • ai/semantic — LLM semantic enrichment
  • ai/core — PDF + DB + masking + serializers
  • ai/api — FastAPI endpoints
  • ai/cli — command-line analysis tools
  • ai/graph — DAG + Markov + governance graphs
  • docs/ — architecture + legal notes

Execution Pipeline

  • PDF ingestion
  • Structured field extraction
  • Database crosscheck
  • Rule validation engine
  • Risk flag detection
  • LLM explanatory enrichment
  • Versioned DAG execution record

Governance Model

  • No automatic decisions
  • Human-in-the-loop validation
  • Versioned analysis graphs
  • Execution hash tracking
  • Legal defensibility
  • Multi-CREA ready
