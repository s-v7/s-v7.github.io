Overview

CREAPI Public Bot is a modular institutional assistant designed to provide normative and procedural guidance for professionals and companies within the CONFEA/CREA ecosystem.

It delivers structured, official-oriented answers based on normative knowledge bases, institutional flows and supervised LLM fallback — without accessing personal or confidential data.

Architecture Layers

  • Intent classification layer
  • Knowledge module routing
  • Normative markdown knowledge base
  • Organogram-based service routing
  • Institutional response formatter
  • LLM normative fallback
  • Telegram notifier integration
  • Event + webhook integration layer

Knowledge Modules

  • knowledge/mod_art — ART guidance
  • knowledge/mod_cat — CAT guidance
  • knowledge/mod_pf — PF registration
  • knowledge/mod_pj — PJ registration
  • knowledge/mod_financeiro — annuities & fees
  • knowledge/mod_fiscal — inspection
  • knowledge_normas/ — normative corpus

Knowledge is structured in Markdown and versionable for governance.

Organogram Routing

Institutional routing maps user topics to CREA departments and divisions, ensuring correct guidance and official channel direction.

  • ART → ART Division
  • CAT → Technical Acervo
  • PF/PJ → Registration Sector
  • Financial → Collection Sector
  • Inspection → Fiscalization Sector

Security Guardrails

  • No personal data lookup
  • No process status disclosure
  • No credential handling
  • Sensitive identifiers blocked
  • Official channel redirection enforced

LLM Usage Policy

  • Used only for complex normative explanation
  • Institutional prompt template
  • No normative invention allowed
  • Must recommend official confirmation when uncertain
